🚧 In Development — Phase 1: Foundation & Infrastructure

## Tech Stack
- **API:** FastAPI
- **Database:** PostgreSQL 16 + pgvector

- **Embeddings:** OpenAI text-embedding-3-small
- **Containers:** Docker + Docker Compose
